# HauseMOTD
A simple, lightweight MOTD manager for Paper/Spigot Minecraft Servers (1.20+).

## Features
- Create custom MOTD messages
- Color code support
- Simple commands
- Permission-based access

## Commands

/motd <message> - sets new MOTD
/motd list - Lists all MOTD's
/motd remove <#> - Remove a MOTD

## Permissions
hausemotd.use - Full access

## Installation

1. Download the plugin jar from GitHub
2. Place it in your server's /plugins folder
3. Restart the server

## Compatability
- Tested on Paper 1.20+
- Backwards compatible with Spigot 1.20+
